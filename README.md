### Development

1.  Clone the repo.
2.  Install dependencies

```bash
yarn install
```

3.  Start the development server

```bash
yarn start
```

4.  Okay, it's going to get a little weird here.
5.  Create a new Dat site in your Beaker Browser library.
6.  Add your Dat url to config.js
7.  Run a build

```bash
yarn run build
```
